# Technical Art Resources Collection

这是一个技术美术（Technical Artist）相关资源的集合仓库，包含各种教程、着色器示例、工具脚本和职业发展指南。

## 📁 项目结构

```
ForAIRes/
├── docs/              # 文档目录
│   ├── tutorials/     # 教程文档
│   ├── technical/     # 技术文档
│   ├── career/        # 职业发展
│   └── ai/            # AI相关文档
├── tools/             # 工具和脚本
│   ├── color/         # 配色工具
│   └── shaders/       # 着色器工具
└── assets/            # 资源文件
    ├── images/        # 图片资源
    └── html/          # HTML文件
```

## 🛠️ 工具和脚本

### 配色生成器
使用方法：
```bash
cd tools/color
python generate_color_palettes.py
```
这个脚本会生成一系列和谐的配色方案，并将结果保存为图片和 HTML 文件。
